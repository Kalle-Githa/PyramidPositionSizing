# Pyramid Position Sizing

---

## About the project

This project is a web application built with **Blazor** and **.NET**.  
The purpose of the application is to calculate position sizing and distribute pyramid orders based on user-defined values such as price levels, risk amount, and number of orders.

It was created to practice C# logic, financial calculations, and dynamic user interaction in a modern .NET web application.

---

## Built with

- **C#**
- **.NET**
- **Blazor**

## Features

- Position sizing calculation
- Pyramid order distribution
- User-defined price levels
- Risk amount input

---

## How to use

1. Enter the entry price and stop loss  
2. Enter the total risk amount  
3. Choose how many pyramid orders to create  
4. Add the price levels for the orders  
5. View the calculated position sizing and order distribution
